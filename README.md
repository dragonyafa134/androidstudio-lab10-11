# Лабораторнная работа 10-11

Приложение для отображения списка студентов с использованием принципов Material Design 3. Каждый студент представлен в виде карточки с аватаркой, именем, возрастом и юмористическим описанием. Реализована поддержка светлой и тёмной темы, кастомная типографика, анимация раскрытия/сворачивания карточек.

## Цель работы

Изучить принципы Material Design в Android-приложениях с использованием Jetpack Compose. В ходе работы были освоены:
- создание цветовых схем для светлой и тёмной темы
- настройка кастомной типографики (шрифты Montserrat и Abril Fatface)
- применение форм и скруглений к компонентам
- работа с компонентами Material Design 3 (Scaffold, TopAppBar, Card, LazyColumn)

## Функциональность

-  Отображение списка из 10 студентов
- ️ Круглые аватарки студентов с правильным кадрированием
-  Возможность раскрытия/сворачивания карточки для просмотра описания
-  Плавная анимация при раскрытии (spring-анимация)
-  Поддержка светлой и тёмной темы (автоматическое переключение)
-  Кастомные скругления углов карточек (левый нижний и правый верхний)
-  Адаптивный интерфейс с использованием Scaffold и TopAppBar

## Технологии

- **Язык**: Kotlin
- **Дизайн-система**: Material Design 3
- **Библиотеки**:
    - `androidx.compose.material3` — компоненты Material Design
    - `androidx.compose.material:material-icons-extended` — расширенный набор иконок
    - `androidx.compose.ui` — основа Compose UI
    - `androidx.compose.foundation` — базовые компоненты (LazyColumn, и др.)
    - `androidx.compose.animation` — анимации (spring)
- **Шрифты**: Google Fonts (Montserrat, Abril Fatface)
